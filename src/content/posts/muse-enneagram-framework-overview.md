# Muse-Enneagram Framework Overview

This is a map of maps. The Muse-Enneagram Framework is not a single document but a constellation of interconnected analyses that together describe a nine-fold architecture operating simultaneously across mythological, biological, celestial, and psychological domains.

If you are encountering this body of work for the first time, this overview provides the topology. It tells you what connects to what, and why the connections matter.

## The Core Matrix

At the center of the framework sits the Muse-Enneagram Matrix — a nine-by-nine correspondence table mapping each Greek Muse to an Enneagram type, an endocrine hormone, a constellation, and a set of behavioral patterns.

The primary correspondences:

1. **Polymnia-Melatonin-Type 1** — The Rhythm Compiler. Sacred hymns meet circadian regulation meet the Reformer's perfectionism. The pattern: precise timing as both creative discipline and biological necessity.

2. **Clio-Oxytocin-Type 2** — The Memory Binder. Historical preservation meets bonding chemistry meets the Helper's relational focus. The pattern: connection as both archival function and hormonal imperative.

3. **Euterpe-Endorphins-Type 3** — The Release Channel. Musical healing meets pain relief meets the Achiever's drive for resolution. The pattern: transformation through the completion of tension.

4. **Thalia-Dopamine-Type 4** — The Anticipation Engine. Comedy meets reward prediction meets the Individualist's longing. The pattern: pleasure as recognition, not possession.

5. **Melpomene-Cortisol-Type 5** — The Awareness Amplifier. Tragedy meets stress response meets the Investigator's intensity. The pattern: suffering as information, not punishment.

6. **Erato-Estrogen-Type 6** — The Creation Cycle. Love poetry meets generative chemistry meets the Loyalist's devotion. The pattern: creation through committed encounter.

7. **Calliope-Testosterone-Type 7** — The Epic Drive. Epic poetry meets forward momentum meets the Enthusiast's ambition. The pattern: sustained effort as creative act.

8. **Terpsichore-Adrenaline-Type 8** — The Movement Protocol. Dance meets emergency response meets the Challenger's physicality. The pattern: action that bypasses thought.

9. **Urania-Serotonin-Type 9** — The Order Function. Astronomy meets balance regulation meets the Peacemaker's equanimity. The pattern: peace as the recognition of underlying geometry.

The core matrix operates on its own. But the framework's real power emerges through overlays — additional pattern systems that align with and amplify the nine-fold base.
